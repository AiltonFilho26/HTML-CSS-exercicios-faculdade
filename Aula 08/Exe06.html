O estilo a:link faz com que os links peguem estas características ( font-style: italic; color: green;) sejam aplicadas. Abra o Exe01

O estilo a:hover faz com que o link pegue esta característica ( font-style: italic; color: navy;) quando passamos o mouse sobre ele. Abra o Exe02

O estilo a:visited faz com que o link pegue a característica ( color: red;) após ser clicado. Abra o Exe03
