Nova janela aberta usando "TARGET"

O atributo "TARGET" informa ao browser onde abrir a página Web que aparece no link do atributo "HREF".

Se você utilizar "_BLANK" para o alvo, o browser sempre abrirá uma nova janela para exibir a página.
